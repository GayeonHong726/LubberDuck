# AI를 활용한 칵테일 비대면주문과 관리 사이트

### 프로젝트 개요
- 손님들에게 편리한 칵테일 비대면 주문과 개인활동 관리 기능을 제공하고, 관리자에게는 컨텐츠 관리 기능을 제공합니다.

### 팀원
- 6명으로 구성
